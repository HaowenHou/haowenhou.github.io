# PersonalHomepage

Static personal homepage for GitHub Pages.

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## GitHub Pages

This repository is intended for `HaowenHou/haowenhou.github.io`.
Enable Pages from the `main` branch. The site will be served at:

`https://haowenhou.github.io/`

The site has no build step. `.nojekyll` is included so GitHub Pages serves assets directly.
