# PersonalHomepage

Static personal homepage for GitHub Pages.

## Local preview

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## GitHub Pages

Use one of these deployment styles:

- For `HawHello.github.io`, put these files at the repository root and enable Pages from the `main` branch.
- For a project repository, keep these files at the repository root and enable Pages from the `main` branch. The page will be served at `https://HawHello.github.io/<repo-name>/`.

The site has no build step. `.nojekyll` is included so GitHub Pages serves assets directly.
